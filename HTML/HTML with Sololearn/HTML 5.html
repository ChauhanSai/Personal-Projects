Page Structure

  • Header- Branding
  • Nav- Site navigation
  • Article- Main content(replaces div)
  • Section- Within Article, Section of main content
  • Aside- Extra information next to Article
  • Footer- Website information

HTML 5 must be denotated with a !DOCTYPE HTML at the start of the file
